# TypeScript - João Faccio

Este repositório contém os exercícios práticos desenvolvidos para a disciplina de **Aplicações Móveis**.

## Conteúdo
O arquivo `typescript-joao-faccio.ts` contém 10 funções e interfaces que cobrem:
* Tipos primitivos e parâmetros opcionais;
* Interfaces e Objetos complexos;
* Generics `<T>`;
* Union Types;
* Tipagem de componentes e funções (void).

## Como rodar a validação
Caso deseje validar a tipagem, utilize o comando:
`npx tsc --noEmit`
